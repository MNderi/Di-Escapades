# Di-Escapades 🌍

A static profile website built as a school team project, showcasing the personal profiles of six individuals.

## About

Di-Escapades is a multi-page static website that highlights the stories and profiles of six real people. It was built collaboratively as part of a school assignment, with a focus on clean layout and presentation.

## Built With

- HTML
- CSS
- JavaScript (minimal)

## Features

- Individual profile pages for each of the six featured people
- Clean, readable layout
- Responsive styling

## Getting Started

No installation needed. Simply open `index.html` in your browser to view the site.

```bash
# Clone the repo
git clone https://github.com/MNderi/Di-Escapades.git

# Open in browser
open index.html
```
